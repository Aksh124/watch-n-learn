<h1 align="center">Contributing</h1>

<h2>Commit</h2>

```
Format: <TYPE> <SHORT_MESSAGE>

    TYPE:

        One or combination of multiple types

        [C] - Change in CSS
        [H] - Change in HTML / JINJA
        [J] - Change in JAVASCRIPT
        [O] - Change in other (.gitignore, .md, etc)
        [P] - Change in PYTHON

        Example:

            [CHJ]

    SHORT_MESSAGE:

        Short description of changes (2 - 6 words)
        Should be in present tense

        Example:

            Improve UI

    Example:

        [CHJ] Improve UI

```
